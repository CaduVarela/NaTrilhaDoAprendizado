Branches e Hospedagem

Branches

São ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • Novas funcionalidades;
  • correção de problemas;
  • alteração de código.

Branch Master ou Main: braço principal de um projeto em um repositório.

Branch Secundária: braço secundário de um projeto em um repositório.

Como utilizar no GitHub:

  1. Acessar seu repositório;
  2. Criar uma nova Branch;
    (o botão fica logo acima dos códigos, como padrão estará a branch main)
  3. Realizar suas alterações nos arquivos de sua nova branch;
    (Obs: é possível criar branchs sobre outras branchs)
  4. Para mesclar a branch secundária em sua branch main, você deve criar um pull request;
  5. Depois, na aba de pull requests, você deve realizar o merge (mesclagem) do pull request que você criou, corrigir possíveis erros e finalizar.

Hospedagem

De maneira geral, os serviços de hospedagem são pagos, porém há um recurso fornecido pelo GitHub que é gratuito, o GitHub Pages.

O GitHub Pages oferece hospedagem gratuita de sites. Basta criar uma conta e um repositório público e a URL será: https://nomeusuario.github.io/repositorio

Como utilizar no GitHub:

  1. Acessar seu repositório (deve ser público);
  2. Acessar as configurações (settings) de seu repositório;
  3. Procurar por GitHub Pages;
  4. Em Source, selecionar uma branch para utilizar no site e, depois, salvar;
  5. Abrir a página web/site com o link disponibilizado nas configurações.

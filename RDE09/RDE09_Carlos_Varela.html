Incorporando Multimídias ao Site

Multimídia é uma forma de comunicação que combina diferentes formas de conteúdo, como textos, áudios, imagens, animações ou vídeos em uma única apresentação.

Tag iframe

A tag <iframe> é usada para incorporar outro documento ao documento HTML.

Sintaxe:

<iframe src="URL" height="altura" width="largura" title="título"></iframe>

Tag video

A tag <video> serve para exibir arquivos de vídeo hospedados em seu servidor.

Sintaxe: 

<video poster="URLImagem" controls>
    <source src="URLVideo" type="video/tipo">
    <source src="URLVideo2" type="video/tipo2">
    /* só é necessário um source, os outros servem como outra opção caso o vídeo anterior não carregue (podem haver mais extra)*/
</video>

Tag audio

A tag <audio> serve para exibir arquivos de áudio hospedados em seu servidor.

Sintaxe

<audio preload="metadata" controls>
    <source src="URL1" type="audio/tipo1">
    <source src="URL2" type="audio/tipo2">
    // só é necessário 1, os outros servem como outra opção caso o vídeo 
    // anterior não carregue (podem haver mais extra)
</audio>

Tag picture

A tag <picture> serve para definir imagens de tamanhos diferentes de acordo com o tamanho da tela. Funciona com responsividade

Sintaxe:

<picture>
    <source media="(max-width:tamanho1 px)" srcset="URL_Imagem_Pequena">
    <source media="(max-width:tamanho2 px)" srcset="URL_Imagem_Media">
    <img src="URL_Imagem_Grande" alt="Descição imagem">
</picture>

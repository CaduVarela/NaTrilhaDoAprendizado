Outras tags HTML

Explicando e exemplificando o uso de algumas outras tags utilizadas em HTML.

Tag table

A tag <table> é uma tag que exige fechamento, ou seja, precisa de </table>, e serve para criar tabelas em HTML.

Para utilizar o table, é necessário saber de duas outras tags importantes, <tr> e <td>, que também exigem fechamento:

  • <tr> → table row → linha da tabela (define linha);
  • <th> → table header → Título/cabeçalho de uma coluna ou linha na tabela;
  • <td> → table data → dado da tabela (define célula).

OBS: A tag <th> pode receber o atributo "scope" para indicar o sentido (direção) que o título aponta, usando o valor "col" para indicar coluna e "row" para indicar linha. Por padrão (se nenhum valor for informado) o atributo é carregado com o valor "col".

Por exemplo, observe o seguinte código:

<table>
    <tr>
        <th>[Coluna1]</th>
        <th>[Coluna2]</th>
        <th>[Coluna3]</th>
    </tr>
    <tr>
        <th>[linha1]</th>
        <td>[dado1]</td>
        <td>[dado2]</td>
    </tr>
    <tr>
        <th>[linha2]</th>
        <td>[dado1]</td>
        <td>[dado2]</td>
    </tr>
</table>

Esse código gera uma tabela como:

[Coluna1] [Coluna2] [Coluna3]
[Linha1] [dado1] [dado2]
[Linha2] [dado1] [dado2]

OBS: sem utilizar CSS

extra

A tag <td> pode receber atributos para ocupar mais de um espaço de dado utilizando o seguinte:

  • <td rowspan="X"></td> → utilizado para ocupar X linhas na tabela;
  • <td colspan="X"></td> → utilizado para ocupar X colunas na tabela.

OBS: X representa um número inteiro e positivo

Observe o seguinte exemplo onde "A" tem rowspan="2" e "F" tem colspan="2"

A B C D
E F

OBS: CSS utilizado para efeitos de demonstração visual

Tags del e ins

As tags <del> e <ins> necessitam fechamento e servem para indicar partes removidas (deletadas) e adicionadas (inseridas), respectivamente. Por padrão, quando utilizado, o texto dentro de del fica riscado no resultado final, ao passo que dentro de ins fica sublinhado.

Vamos ao exemplo, observe o seguinte código:

<p>Essa parte foi <del>removida</del> </ins>adicionada</ins></p>

Isso gera o seguinte resultado:

Essa parte foi removida adicionada

Tag sub e sup

As tags <sub> e <sup> necessitam fechamento e servem para elevar e abaixar um elemento na linha, respectivamente. Observe o código a seguir:

<p>A<sub>1</sub> = 5<sup>2</sup></p>

Esse código gera o seguinte resultado:

A1 = 52

Tags q, blockquote e cite

As tags <q>, <blockquote> e <cite> necessitam fechamento e servem todas para fazer sitações, mas cada uma com suas particularidades:

  • <q> → (quote) fazer uma citação comum, geralmente curta;
  • <blockquote> → citar uma informação já citada vinda de alguma fonte externa; (geralmente mais longas que as quotes);
  • <cite> → citar uma obra (livro, pintura, música, etc.).

observe os seguintes exemplos de q, blockquote e cite, respectivamente:

  1. <p><q>Só sei que nada sei</q> ~Sócrates</p>

  2. <p>Informação citada no site do Tecnoblog:</p>
    <blockquote cite="https://tecnoblog.net/">Lorem ipsum dolor sit amet consectetur adipisicing elit. Laborum deserunt obcaecati ducimus repellendus magnam maxime facilis culpa et veritatis quidem esse, iusto recusandae.</blockquote>

  3. <p><cite>O grito</cite> foi um quadro pintado por Edward Munch em 1893.</p>

OBS: O cite usado como atributo é usado para referenciar de onde vem a informação.

Resultados, em ordem:

  1. Só sei que nada sei ~Sócrates

  2. Informação citada no site do Tecnoblog:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Laborum deserunt obcaecati ducimus repellendus magnam maxime facilis culpa et veritatis quidem esse, iusto recusandae.

  3. O grito foi um quadro pintado por Edward Munch em 1893.

Tag abbr

A tag <abbr> necessita fechamento e serve para dar uma "descrição" para uma sigla quando o usuário passar o mouse em cima dela. Para indicar essa descrição, é utilizado o atributo title, observe o código de exemplo a seguir:

<p>A <abbr title="Organização das Nações Unidas">ONU</abbr> foi formada em 1945.</p>

Resultado do código:

A ONU foi formada em 1945.

Tag address

A tag <address> necessita fechamento e serve para definir as informações de contato para o autor/dono de um documento ou artigo.

Observe o seguinte código de exemplo:

<address>
Escrito por <a href="mailto:joaozinho@exemplo.com">Joãozinho Pereira</a>
Telefone para contato: +55 42 912345678
Cidade da Jabuticaba, Brasil
</address>

Confira o resultado do código:

Escrito por Joãozinho Pereira
Telefone para contato: +55 42 912345678
Cidade da Jabuticaba, Brasil

Tag code

A tag <code> necessita fechamento e serve para escrever trechos de código na página, confira o exemplo no código a seguir:

<p>A tag <code>&lt;br&gt;</code> é utilizada para quebrar linhas em HTML</p>

Confira o resultado do código:

A tag <br> é utilizada para quebrar linhas em HTML

Tag pre

A tag <pre> necessita fechamento e serve para escrever um trecho com espaços predefinidos, observe no código de exemplo

<pre>
Lorem ipsum     dolor     sit amet,
         consectetur 
adipisicing        elit.
</pre>

Confira o resultado do código:

Lorem ipsum     dolor     sit amet,
         consectetur 
adipisicing        elit.

OBS: Observe que os espaços que estavam no código não "somem" no resultado final, como deveria ocorrer normalmente (sem a tag pre)

Tag bdo

A tag <bdo> necessita fechamento e serve para indicar uma direção para um texto. Para indicar essa direção é utilizado o atributo dir, que pode receber dois valores (direções):

  • ltr → left to right → esquerda para a direita;
  • rtl → right to left → direita para a esquerda.

Observe o código de exemplo:

<p><bdo dir="rtl">Esse texto vai ficar escrito da direita para a esquerda</bdo></p>

Confira o resultado do código:

Esse texto vai ficar escrito da direita para a esquerda

Tag kbd

A tag <kbd> necessita fechamento e serve para indicar/definir uma tecla do teclado.

Veja o exemplo em código:

<p>No Windows você pode utilizar <kbd>Ctrl</kbd> + <kbd>C</kbd> para copiar um texto, imagem ou arquivo.</p>

Confira o resultado do código:

No Windows você pode utilizar Ctrl + C para copiar um texto, imagem ou arquivo.
